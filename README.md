# 💬 Chat With PDF Web

A simple and user-friendly web application that allows users to **upload PDF documents** and **interact with them using AI**. This project is built with a clean frontend interface and integrates with an AI backend to process and chat with the uploaded PDFs.

## 🚀 Features

- 📄 Upload your PDF file
- 🤖 Chat with the content of your PDF using AI
- ⚡ Fast and interactive responses
- 🎨 Simple and intuitive UI/UX

## 🖥️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Hosting**: [InfinityFree](https://infinityfree.net/)
